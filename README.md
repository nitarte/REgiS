# REgiS

Плагин на регистрацию для хостингов по типу про хоста, которые дают домен и порты.

Плагин нужен тем, кто хочет чего-то нового — не обычные старые ваши /reg /log, а что-то реально крутое. Я не встречал таких плагинов и не обещаю, что этот выдаст что-то нереальное, но как форс подобных — пойдет.

**Технические детали:**

  * **SQLite** — база данных находится прямо в папке плагина, никакой лишней возни с MySQL.
  * **Перспективы** — тема зашла, поэтому в 1.4 я выкатил мощное обновление безопасности и фронтенда.

-----

### Безопасность плагина:

  * **Бэкдоры** — я старался не допускать таких ошибок, но наши великие пиксельные войны полюбому их найдут.
  * **Хэш паролей** — теперь используется **BCrypt**. Даже если базу украдут, пароли расшифровать не смогут.
  * **Защита сессий** — внедрены временные токены. Никто не сможет залезть в ваш профиль, просто подставив ник в ссылку.
  * **RegisDefA** — моя система защиты от ботов. Есть 3 режима (MAX, AVERAGE, WEAK). Кикает за спам, высокий пинг и кривые ники.
  * **Фильтр стран** — можно настроить вход только для определенных стран (белый/черный список по IP).
  * **Discord 2FA** — если привязать дискорд, при входе бот будет кидать код в личку.
  * **Админ аккаунты** — защита на уровне сессий и токенов, так что просто так не взломать.

-----

### Че это такое вообще?

Представим, есть хостинг, домен и порт. Вы вводите их в конфиг и получаете быстрый, удобный сайт для регистрации. Главное — иметь открытый порт для сайта.

### Функционал и возможности:

1.  **Интерактивный Web-интерфейс** с полноценной поддержкой кастомных доменов и портов.
2.  **Бесшовная авторизация** — автоматическое сопоставление игрового ника с веб-профилем через защищенные токенизированные URL.
3.  **Session Persistence** — технология локального хранения данных в браузере для исключения повторного ввода пароля.
4.  **Discord OAuth2 Integration** — безопасная привязка аккаунта с защитой от перехвата (доступна строго после успешного логина).
5.  **Web-Control Panel** — удаленное управление игровым аккаунтом прямо из браузера:
      * **kick / ban / unban** — оперативное управление доступом.
      * **reset** — защищенный протокол смены пароля.
      * **ips** — аудит всех сетевых подключений к аккаунту.
6.  **Smart IP Auth** — интеллектуальный автовход по IP с настраиваемым временем жизни сессии.
7.  **Auth Lockdown System** — визуальная изоляция неавторизованных (эффекты ослепления и блокировка перемещения).
8.  **Dynamic UI Engine** — полная кастомизация визуала сайта через замену `background.jpg`.
9.  **Universal Localization** — гибкая настройка всех системных уведомлений через `config.yml`.
10. **State Recovery** — гарантированный возврат игрового режима (GameMode) и координат после завершения авторизации.
11. **Administrative Tools** — добавлена команда `/unregban` для мгновенной разблокировки игроков, попавших под санкции RegisDefA.
